1 ワイヤーフレーム作成方法

作成ツール
【使用したプロンプト】

あなたは、ハイエンド体験型ブランドのUXアーキテクト兼プロダクトデザイナーです。
以下の条件に基づき、ECサイトの「トップページ」と「デジタル調香ページ」のWireframeを設計してください。
出力形式は、HTMLドキュメント内にSVGとして埋め込む形にしてください。

------------------------------------------------

・固有商品名は使わない
・サイトマップにあるカテゴリ名のみ使用する

------------------------------------------------
■UX思想
------------------------------------------------
調香は、たんなる組み合わせの「診断」ではない。
ユーザーが、「実際に自分で香りが変わっていくことを体験」する。

UIは、
・直感的
・詩的
・科学っぽくしすぎない
ところに留意する。

■出力
------------------------------------------------

2 トップページ

トップページ ワイヤーフレーム（PC版） 必須
references/05-wireframe の SVG アセットを配置
配置意図の説明 必須
各セクションの配置理由とユーザーへの効果

配置意図の説明

  • 現行TOPの役割を維持：回遊（ブランド / アイテム / 香りから探す / 記事 / Sale / 実店舗）を崩さず、上段に「体験」導線を追加。
  • 体験導線は一点集中：ファーストビューで「香りと遊ぶ！香游」を強く提示し、直帰を抑制。
  • 理解補助：「体験の流れ（3ステップ）」で、ユーザーが“何が起きるか”を即理解できるようにする。
  • 再掲で回収：スクロール後にも体験バナーを配置し、商品閲覧から体験への転換を促す。

3 下層ページ

下層ページ ワイヤーフレーム１（デジタル調香ページ） 必須
各下層ページのワイヤーフレームを順に配置

【香りと遊ぶ！香游（デジタル調香ページ）】

配置意図の説明

  • 単一目的の没入ページ：色×粒子×サウンド（任意）で“混ぜる”体験に集中させる。
  • 操作の即時理解：初回ヘルプで「スクロール＝レイヤー」「ドラッグ＝広がり」「色＝ノート」を最短で理解。
  • 次行動の自然さ：結果表示の直後に「実店舗体験へ」。香りが生まれた瞬間の動機で遷移させる。
