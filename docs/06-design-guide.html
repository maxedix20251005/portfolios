1 デザインコンセプト

デザインキーワード 必須
デザインの方向性を表すキーワード3つとその説明

1. 一滴を主役にする導線
理由: トップページからワークショップページへ、離脱されることなく迷わず到達させることが、短期成果（到達率改善）の最優先だから。
表現方法: ファーストビューで体験価値を主役化、固定/中間CTAを明確化、視線誘導を阻害しない余白設計。

2. 安心感が決断を後押しする設計
理由: CVR改善には「魅力」だけでなく「不安解消」が必要だから。
表現方法: 価格・所要時間・初心者歓迎・持ち物不要・FAQを上部〜中段に一目で見やすく整理し、レビュー抜粋で信頼を補強する。

3. 記憶に残る体験価値の可視化
理由: 体験そのものの強み（高評価）を、Web上で瞬時に伝える必要があるから。
表現方法: 完成イメージ写真、体験ステップ、情緒的コピーを一貫配置し、「自分ごと化」しやすいストーリー構成にする。

5 レイアウトルール

レイアウト基本ルール 必須
グリッド・余白・角丸・シャドウ等の数値ルール

1. コンテンツ最大幅
PC: 最大 1200px（左右パディング 40px）
タブレット: 最大 960px（左右パディング 24px）
スマートフォン: 最大 100%（左右パディング 16px）

2. グリッドシステム（カラム数・ガター幅）
PC: 12カラム / ガター 24px
タブレット: 8カラム / ガター 20px
スマートフォン: 4カラム / ガター 16px

3. セクション間の余白
PC: 96px
タブレット: 72px
スマートフォン: 56px

4. 要素間の余白
大要素（カード群・主要ブロック間）: 32px
中要素（見出しと本文・フォーム行間）: 20px
小要素（ラベルと入力欄・補足テキスト）: 8px

5. 角丸の数値
カード・モーダル: 12px
ボタン・入力欄: 10px
タグ・バッジ: 8px

6. シャドウ設定（box-shadow）
基本カード: 0 6px 18px rgba(44, 44, 44, 0.10)
強調カード/ホバー: 0 12px 28px rgba(44, 44, 44, 0.14)
フォーカスリング: 0 0 0 3px rgba(44, 44, 44, 0.14)

7. ブレークポイント（PC / タブレット / スマートフォン）
PC: 1024px以上
タブレット: 768px〜1023px
スマートフォン: 767px以下
